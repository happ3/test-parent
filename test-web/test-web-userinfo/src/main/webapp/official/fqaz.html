常见问题
   为什么有的新任务没法接？
   部分任务只限首次，之前已经下载过，在本平台将无法接受该任务。
   为什么有的任务完成了但是没有到账？
   由于有些任务需要进行后台验证，比如是否进行试玩、是否注册等，对于后台验证可能会存在延时的情况。
   为什么任务一直在进行中？
   请先检查是否按照要求完成详情页内标示各项，比如注册，打开，试玩3分钟；搜猴儿独创一键式任务下载需要你保持开启状态。
   申请提现了，为什么一直没收到钱？
   请先检查您的账号是否设置正确，提现审核一般会在24小时内完成，节假日会存在延时，请耐心等待。
   任务什么时候更新？
   任务的上架没有固定时间段，大部分集中在15点-22点，任务集中时段我们会有推送通知，建议您不要关闭搜猴儿。
